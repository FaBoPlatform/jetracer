# FaBo JetRacer Kit

このレポジトリは、[NVIDIA-AI-IOT/jetracer](http://github.com/NVIDIA-AI-IOT/jetracer)のForkプロジェクトです。

教育現場への導入を目的に、株式会社FaBoがキット化し、改良したバージョンとなります。　

主な修正点としては

- Jetson Nanoをマウントする箇所のカーボン化
- TT02向けに細かい値の設定を可能にしたノートブック
- 学習方法等の工夫をおこなったノートブック

などが挙げられます。
